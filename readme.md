## Favorite-this demo (with ajax)

This is a modified version of the Laracasts lessons [favorite-this pt1](https://laracasts.com/lessons/favorites-part-1) and [favorite-this pt2](https://laracasts.com/lessons/favorites-part-2) with source at [favorite-this source](https://github.com/laracasts/Favorite-This-Demo). This version is spiced with ajax.
See working [demo](http://favorite-ajax.eu1.frbit.net) and enjoy.


## Use cases

1. Favorite anything buttons.
2. Follow buttons.
3. Like buttons, etc.
